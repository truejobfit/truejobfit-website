What's New

Release notes

v2.9.0 April 2026
Contract roles now show day rates instead of annual salary (e.g. A$850–1,050 / day).

ATS scanner: required vs preferred keyword badges, weighted hard skill scores, improved keyword filtering, full copy report.

Gap coach: Copy all gaps button compiles all reframes into one clipboard-ready summary.

LinkedIn job page panel now renders correctly above "About the job" on direct job links.
v2.7.0 March 2026
Security hardening: CORS restricted to extension only, model pinning prevents proxy abuse, rate limiting on CV parsing, message sender validation added.

Four-tier score colours: green (85+, Strong Match), blue (75-84, Good Match), orange (65-74, Stretch Role), red (below 65, Weak Match). Consistent across panel, dashboard, badges, and score map.

Interview prep regenerate fix: refreshed questions now use full CV context, eliminating hallucinated company names and fabricated metrics. Added hallucination prevention rules and logistics question filtering.

Dashboard comparison mode: expand multiple jobs simultaneously with Expand All/Collapse All toggle. Score map uses beeswarm jitter layout for overlapping dots.

Gap coach improvements: visual separators between gaps with numbered labels. Reframe All button triggers all gap reframes at once. Auto-opens gap section when clicked.

Cover letter tone improvements: fixed presumptuous closing paragraphs across all tones. Softened Confident tone. Improved opening framing. Added new Formal tone option for traditional industries.

Popup: API key removal button for easy switching between BYOK and free credits.
v2.6.0 March 2026
Try 3 jobs free without an API key. New users can now scan 3 LinkedIn jobs with full analysis, cover letters, and interview prep before setting up their own OpenAI key. Data routes through a secure proxy - your resume and job data are never stored or logged. After 3 free analyses, add your own key for unlimited use. $5 covers 200+ scans.
v2.5.0 March 2026
Interview prep overhaul. Questions and answers now reference the actual company, not a generic version of the role — company context extracted from the LinkedIn sidebar and job description. Recruitment agency listings detected automatically so the recruiter's company info isn't mistaken for the hiring company. Forward-looking questions ("first 90 days") now produce Listen/Assess/Act answers grounded in the target company. Question count calibrated to interview format: phone screens get 3–4 concise questions, hiring manager rounds get 5–6 deep dives. Each question ranked by probability (Likely, Possible, Less likely). STAR stories now include realistic tension and decision points, not just CV bullet reformatting. Questions to ask the interviewer capped at 15 words and written conversationally. Practical notes surface office requirements, travel, visa restrictions, and other logistics from the job description.
v2.4.0 March 2026
Overqualification UI overhaul: when a role asks for significantly less experience than you have (2.0x+ ratio), the score label changes from "Strong Match" to "Overqualified" in amber, and a prominent non-dismissable banner appears above the score explaining the level mismatch. The score number stays accurate for skills match, but the label and banner ensure you can't miss the level-fit signal. Overqualification prompt penalties tightened across all tiers. Narrowed extension permissions for tighter Chrome Web Store compliance.
v2.3.0 March 2026
Works on all LinkedIn job page layouts — added support for a newer LinkedIn DOM structure that was silently preventing the panel from appearing for some users. Version indicator added to popup with link to release notes. ATS keyword concentration warnings flag when too many edit suggestions target the same CV engagement. Updated What's New banner.
v2.2.0 March 2026
Fixed panel not appearing on LinkedIn search pages. The MutationObserver now correctly recovers when LinkedIn's right-pane DOM takes time to settle.
v2.1.0 March 2026
ATS Keyword Check: on-demand modal that extracts 15–25 keywords from every job description, classifies each against your resume as found, wording gap, or genuine gap. Wording gaps include exact edit instructions — which CV bullet to change and what word to swap. One-click copy per item and a copy-all cheat sheet grouped by action type. Also fixed applicant count extraction, location and work type detection, and overqualification field handling.
v2.0.1 March 2026
Cache Browser improvements: delete individual cached entries, potential score now always shown in detail rows.
v2.0.0 March 2026
Major release. Score badges on LinkedIn search pages. Scanned Jobs browser with sort, search, backfill, and inline editing. Interactive demo for 5 role types. First-run onboarding with role selector and demo preview. Potential score and gap type tags on every analysis. Dashboard gap coach. Analysis caching (14-day TTL). Score map visualisation. Keyboard shortcuts. CSV export. Cover letter and interview prep downloads. Weekly usage stats. Rating prompt. What's New banner. Locked to GPT-4.1.
v1.0.0 March 2026
Initial release. Match scoring with classification-to-band system. Salary estimates. Company insights. Gap Coach with bridge language. Cover letter generator (4 styles, 3 lengths). Interview prep with STAR stories. Application tracker and dashboard.
