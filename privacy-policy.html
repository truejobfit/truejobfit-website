Privacy Policy

Last updated: March 2026

True Job Fit is a Chrome extension that scores how well your CV matches LinkedIn job listings. This policy explains what data the extension handles, where it goes, and what it does not do.

Data storage

True Job Fit stores all data locally in your browser using chrome.storage.local. This includes:

  • Your parsed CV profile and raw CV text
  • Your OpenAI API key
  • Tracked jobs, application statuses, and notes
  • Analysis cache (cached match results for previously viewed jobs)
  • Cover letters and interview prep data
  • Usage statistics (analysis count, weekly stats) — stored locally only
  • Extension preferences (panel state, cover letter preferences)

None of this data is transmitted to any server controlled by True Job Fit. There is no True Job Fit backend, database, or cloud storage.

API calls to OpenAI

When you analyse a job listing, generate a cover letter, request gap coaching, or generate interview prep, True Job Fit sends your CV profile and the job description text to OpenAI (api.openai.com), subject to OpenAI's privacy policy. How this works depends on whether you have configured your own API key:

If you have your own OpenAI API key (BYOK mode): All requests go directly from your browser to OpenAI using your key. True Job Fit does not proxy, log, or intercept these calls. Your API key is stored only in your browser's local storage and is never sent to any True Job Fit server.

If you are using free credits (no API key): Up to 3 job analyses are routed through a secure proxy server (tjf-proxy.truejobfit.workers.dev) operated by the developer, which forwards the request to OpenAI using a server-side key. See "Free credit proxy" below for full details.

In both cases, requests contain your CV profile text and the job description text necessary for analysis. They do not contain your name, email address, or any payment information.

No analytics, tracking, or telemetry

True Job Fit does not use any external analytics services, telemetry, crash reporting, or usage tracking. No data about how you use the extension is sent to any server. Usage statistics (such as the number of jobs analysed per week) are stored locally in your browser for your own reference and are never transmitted externally.

No advertisements

True Job Fit does not display advertisements and is not affiliated with any advertising network.

Free credit proxy

True Job Fit offers up to 3 free job analyses for new users who have not yet set up their own OpenAI API key. These analyses are routed through a proxy server (tjf-proxy.truejobfit.workers.dev) operated by the developer and hosted on Cloudflare Workers.

  • The proxy forwards your request to OpenAI and returns the result. It does not store, log, or retain your CV text or job description content.
  • A random anonymous identifier (UUID) is stored in your browser and sent with each request solely to count how many free analyses have been used. This identifier is not linked to your name, email, or any other personal information.
  • A credit counter (0–3) is maintained in Cloudflare KV storage against your anonymous UUID. No other data is stored server-side.
  • Once you add your own OpenAI API key, the proxy is never contacted. All requests go directly from your browser to OpenAI.

Third-party services

External services True Job Fit may communicate with:

  • OpenAI API (api.openai.com) — all AI analysis, cover letter, and interview prep calls. Subject to OpenAI's privacy policy.
  • True Job Fit proxy (tjf-proxy.truejobfit.workers.dev) — free credit users only, as described above. Not contacted once you have your own API key.

No other third-party services are used. There are no advertising networks, analytics services, or data brokers involved.

LinkedIn page content

True Job Fit reads the content of LinkedIn job listing pages in your browser to extract the job description, title, company name, and location. This content is used locally for analysis and is sent to OpenAI as described above. True Job Fit does not collect LinkedIn data in bulk, automate any actions on LinkedIn, or collect data from pages other than the job listing you are currently viewing.

Data deletion

You can delete all locally stored data at any time by:

  • Clicking Clear Profile in the extension popup to remove your CV data
  • Clicking Clear Cache in the extension popup to remove cached analyses
  • Removing the extension from Chrome, which deletes all stored data automatically

Children's privacy

True Job Fit is not directed at children under 13 and does not knowingly collect data from children.

Changes to this policy

If this policy is updated, the revision date at the top of this page will be changed. Continued use of the extension after changes constitutes acceptance of the updated policy.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
