Privacy Policy

Last updated: March 2026

True Job Fit is a Chrome extension that scores how well your CV matches LinkedIn job listings. This policy explains what data the extension handles, where it goes, and what it does not do.

Data storage

True Job Fit stores all data locally in your browser using chrome.storage.local. This includes:

  • Your parsed CV profile and raw CV text
  • Your API key (OpenAI or Anthropic)
  • Tracked jobs, application statuses, and notes
  • Analysis cache (cached match results for previously viewed jobs)
  • Cover letters and interview prep data
  • Usage statistics (analysis count, weekly stats) — stored locally only
  • Extension preferences (model selection, panel state, cover letter preferences)

None of this data is transmitted to any server controlled by True Job Fit. There is no True Job Fit backend, database, or cloud storage.

API calls to AI providers

When you analyse a job listing, generate a cover letter, request gap coaching, or generate interview prep, True Job Fit sends your CV profile and the job description text to the AI provider you have selected. True Job Fit supports two providers:

  • OpenAI — requests are sent to api.openai.com, subject to OpenAI's privacy policy
  • Anthropic — requests are sent to api.anthropic.com, subject to Anthropic's privacy policy

These requests:

  • Use your own API key — True Job Fit does not proxy, log, or intercept these calls
  • Are made directly from your browser to the provider's servers
  • Contain your CV profile text and the job description text necessary for analysis
  • Do not contain your name, email address, or API key in the request body

True Job Fit does not store, transmit, or have access to your API key on any external server. Your API key is stored only in your browser's local storage.

No analytics, tracking, or telemetry

True Job Fit does not use any external analytics services, telemetry, crash reporting, or usage tracking. No data about how you use the extension is sent to any server. Usage statistics (such as the number of jobs analysed per week) are stored locally in your browser for your own reference and are never transmitted externally.

No advertisements

True Job Fit does not display advertisements and is not affiliated with any advertising network.

Third-party services

The only external services True Job Fit communicates with are:

  • OpenAI API (api.openai.com) — when OpenAI is selected as your AI provider
  • Anthropic API (api.anthropic.com) — when Anthropic is selected as your AI provider

Communication occurs only when you explicitly trigger an analysis, cover letter generation, gap coaching reframe, or interview prep generation. No other third-party services are used.

LinkedIn page content

True Job Fit reads the content of LinkedIn job listing pages in your browser to extract the job description, title, company name, and location. This content is used locally for analysis and is sent to your selected AI provider as described above. True Job Fit does not scrape LinkedIn at scale, automate any actions on LinkedIn, or collect data from pages other than the job listing you are currently viewing.

Data deletion

You can delete all locally stored data at any time by:

  • Clicking Clear Profile in the extension popup to remove your CV data
  • Clicking Clear Cache in the extension popup to remove cached analyses
  • Removing the extension from Chrome, which deletes all stored data automatically

Children's privacy

True Job Fit is not directed at children under 13 and does not knowingly collect data from children.

Changes to this policy

If this policy is updated, the revision date at the top of this page will be changed. Continued use of the extension after changes constitutes acceptance of the updated policy.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
